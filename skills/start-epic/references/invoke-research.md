# Invoke Research

*Reference for **[start-epic](../SKILL.md)***

---

Invoke the [technical-research](../../technical-research/SKILL.md) skill for epic research.

```
Research session for: {work_unit}
Work type: epic

Initial context from epic interview:
{compiled epic context from gather-epic-context}

Uncertainties to explore:
{list of unknowns identified in route-first-phase}

Create research file: .workflows/{work_unit}/research/{work_unit}.md

Invoke the technical-research skill.
```

When a research topic is parked as discussion-ready, the processing skill will offer pipeline continuation via workflow-bridge.
